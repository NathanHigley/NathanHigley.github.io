SESSION | INTERNET COMMUNICATION

Open Systems Interconnection Model - Layer 5


What does it do?

The session layer provides the mechanism for opening, closing and managing a session between end-user application processes. Communication sessions consist of requests and responses that occur between applications.

Where does it do this?

Session-layer services are commonly used in application environments that make use of remote procedure calls (RPCs).



How does this work with other layers?

Within the service layering semantics of the OSI network architecture, the session layer responds to service requests from the presentation layer and issues service requests to the transport layer.
